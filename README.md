
System Interaction Overview

This application provides an interactive 3D environment with basic object manipulation, camera control, and visualization modes.

Input Controls

[Left Mouse Button]
Used for direct interaction with scene elements:
Select and drag round handles
Press interactive buttons
Place objects within the environment

[Right Mouse Button]
Controls camera orientation:
Click and drag to rotate the view around the scene

[Tab]
Toggles between Floor Plan Mode which provides an Top-Down view of the environment
and standard 3D-mode
Note: Floor Plan Mode is currently under development and may be incomplete

[P]
Enters the Paint edit mode where changes can be made to the active paints used in the project
Current Limitations
Paint edit mode is in an early development stage and may lack full functionality
Some interactions and visual feedback may not yet be fully implemented
Notes

This system is intended for prototyping and iterative development. Behavior and controls may change as new features are introduced.
